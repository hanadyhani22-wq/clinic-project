# نظام إدارة العيادة الجلدية (Dermatology Clinic System)

---

## 📝 وصف المشروع
نظام ويب متكامل لإدارة العيادات الجلدية ومتابعة المرضى، يتيح إمكانية تسجيل المرضى الجدد، وإدارة سجلاتهم الطبية، مع لوحة تحكم متكاملة للمسؤولين لتنظيم العمل الطبي وحذف أو تعديل البيانات بسهولة وسلاسة.

---

## 💡 فكرة المشروع
تقوم فكرة المشروع على تسهيل العمل الإداري داخل العيادة والاستغناء عن السجلات الورقية من خلال تقديم:
* لوحة تحكم إدارية: لعرض المرضى وإدارة بياناتهم (إضافة، تعديل، حذف).
* الملفات والسجلات الطبية: تتبع التاريخ المرضي لكل مريض والزيارات.
* نظام حماية وصلاحيات: التمييز بين حسابات الإدارة وحسابات المرضى.
